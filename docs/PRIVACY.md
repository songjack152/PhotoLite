# PhotoLite 隐私说明

PhotoLite 是本地优先的照片清理工具。

## 数据处理

- App 读取照片只用于本机展示、筛选、分享和删除确认。
- App 不上传照片、缩略图、位置信息或 EXIF 信息到服务器。
- 当前项目没有后端服务。

## 权限用途

- 照片权限：读取用户授权的照片，展示照片信息，并在用户确认后请求系统删除。
- 文件夹权限：桌面文件夹模式下，只访问用户主动选择的文件夹。

## 删除策略

- 用户通过手势把照片标记为删除。
- 每组结束后必须进入二次确认页。
- 用户确认后，App 调用系统能力执行删除。
- 如果系统删除确认被取消，App 不应把它当成应用错误展示给用户。

## 开源仓库注意事项

请不要提交：

- 本地照片或测试素材
- 构建产物，例如 APK、DMG、IPA、Archive
- Apple / Android 签名证书、Provisioning Profile、keystore
- `.env`、`local.properties` 等本机配置文件
